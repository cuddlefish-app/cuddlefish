## Getting Started

Requisite environment variables:

- `HASURA_HOST`
- `HASURA_PORT`
- `HASURA_GRAPHQL_ADMIN_SECRET`
- `API_SECRET`
- `SENDGRID_API_KEY`
- `GITHUB_API_TOKEN`

Run `npm run gql:codegen -- --watch` in development to generate GraphQL types.

Run the development server:

```
npm run dev
```
